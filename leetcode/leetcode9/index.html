Javascript(Leetcode#9) Palindrome Number

發表於 2022-07-29 更新於 2022-08-30 分類於 Leetcode ， Leetcode Easy 閱讀次數： Disqus：

Difficult:Easy

題目

Given an integer x, return true if x is palindrome integer.

An integer is a palindrome when it reads the same backward as forward.

For example, 121 is a palindrome while 123 is not.

翻譯

給定一個整數 x，如果 x 是回文整數，則返回 true。

當一個整數向後讀和向前讀一樣時，它就是一個回文數。

例如，121 是回文，而 123 不是。

1.小於0直接回傳false
2.建立一個右和左變數，兩個變數依序往前和往後比較
3.若有不相同時直接返回false,都相同時返回true
4.數字為偶數時，只需比到相差為1
5.奇數時，需比較相同位置
